# PVC Operation

### List

The list of fields available to get the persistent volume claim in particular namespace:
- Namespace: Required field

No filed is required to list the persistent volume claim in all namespace.

### Get

The list of fields available to get the persistent volume claim:
- Name: Required field
- Namespace: Required field

### Delete

The list of fields available to delete the persistent volume claim:
- Name: Required field
- Namespace: Required field

### Update

The list of fields available to update the persistent volume claim:
- Name: Required field
- Namespace: Required field
- Size: Required field

### Create

The list of fields available to create the persistent volume claim:
- Name: Required field
- Namespace: Required field
- Size: Required field
- StorageClass: Required field
- AccessMode: Optional field
